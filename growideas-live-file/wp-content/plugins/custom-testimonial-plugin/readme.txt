A custom plugin by GROWIDEAS Innovations to manage and display testimonials with ease.

== Description ==
Custom Testimonial Plugin provides a form for users to submit testimonials and a slider to display them on your site.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-testimonial-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcodes provided to display the testimonial form and slider.

== Usage ==
- **[testimonial_slider]**: Use this shortcode to display the testimonial slider on any page or post.
- **[testimonial_form]**: Use this shortcode to display the testimonial submission form on any page or post.

== Frequently Asked Questions ==
= How do I display testimonials? =
Use the `[testimonial_slider]` shortcode to show testimonials in a slider format.

= How do users submit testimonials? =
Use the `[testimonial_form]` shortcode to display the testimonial form.
